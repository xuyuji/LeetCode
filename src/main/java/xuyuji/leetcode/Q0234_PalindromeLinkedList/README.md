# 234. 回文链表

请判断一个链表是否为回文链表。

**示例 1:**

```
输入: 1->2
输出: false
```

**示例 2:**

```
输入: 1->2->2->1
输出: true
```

**进阶：**
你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题？

 


# 解题思路

* Solution1：递归，递归方法加入计数值，递归到底时计数值即是长度，计算出中间点，然后后半段数入栈，前半段数出栈比对。

  PS：这个方案无法通过leetcode的全部测试用例，当链表很长时调用栈溢出StackOverflowError，这里记录一种思路。

* Solution2：转化为list，然后双指针遍历。
