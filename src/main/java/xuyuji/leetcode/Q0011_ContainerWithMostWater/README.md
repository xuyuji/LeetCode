# 盛最多水的容器

给定 *n* 个非负整数 *a*1，*a*2，...，*a*n，每个数代表坐标中的一个点 (*i*, *ai*) 。画 *n* 条垂直线，使得垂直线 *i* 的两个端点分别为 (*i*, *ai*) 和 (*i*, 0)。找出其中的两条线，使得它们与 *x* 轴共同构成的容器可以容纳最多的水。

**注意：**你不能倾斜容器，*n* 至少是2。


# 解题思路

* Solution1：从两头开始算起，最大的桶底，桶壁收缩。

  收缩思路：较高的那一侧对水位线没有影响，移动不可能增加水位线高度，只可能降低水位线高度，再加上移动带来的桶底长度变小，移动较高一侧的桶壁只会造成容量变小。所以一直移动较低侧桶壁直至桶底长度为1，取移动过程中最大容量为结果。
